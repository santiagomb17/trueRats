Make sure you have downloaded the Plague Rats Repository Zip file aka Team 2 project code zip
Visit https://godotengine.org/download/windows
Download the Standard Version of Godot 64bit/32bit depending on your OS
Launch Godot
On the top right click on a button that says import
Click on Browse and navigate to where the Plague Rats Repository Zip is located 
Click on Plague Rats folder and in that folder select the project.godot file then click import
Click the F5 button and this will run the project app
